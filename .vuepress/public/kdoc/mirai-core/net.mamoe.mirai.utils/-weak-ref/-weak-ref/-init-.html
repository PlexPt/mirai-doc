WeakRef(referent: T)

Weak Reference. On JVM, it is implemented as a typealias referring to WeakReference from JDK.

Details: On JVM, instances of objects are stored in the JVM Heap and are accessed via references. GC(garbage collection) can automatically collect and release the memory used by objects that are not directly referred by any other. WeakRef is not a direct reference, therefore it doesn't hinder GC.
