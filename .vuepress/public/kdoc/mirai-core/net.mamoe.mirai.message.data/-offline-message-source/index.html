OfflineMessageSource

@SinceMirai("0.33.0") abstract class OfflineMessageSource : MessageSource

由一条消息中的 QuoteReply 得到的 MessageSource. 此消息源可能来自一条与机器人无关的消息. 因此无法提供对象化的 sender 或 target 获取.
