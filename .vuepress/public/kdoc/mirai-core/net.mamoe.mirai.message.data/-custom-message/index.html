CustomMessage

@SinceMirai("0.38.0") sealed class CustomMessage : SingleMessage

自定义消息

它不会显示在消息文本中, 也不会被其他客户端识别. 只有 mirai 才能识别这些消息.

目前在回复时无法通过 originalMessage 获取自定义类型消息
