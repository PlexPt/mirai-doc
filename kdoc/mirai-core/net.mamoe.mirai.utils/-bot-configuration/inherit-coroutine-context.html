inheritCoroutineContext

@SinceMirai("0.38.0") suspend fun inheritCoroutineContext(): Unit

使用当前协程的 coroutineContext 作为 parentCoroutineContext

用例:

coroutineScope {
  val bot = Bot(...)
  bot.login()
} // coroutineScope 会等待 Bot 退出
