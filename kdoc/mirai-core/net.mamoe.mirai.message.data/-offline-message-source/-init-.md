# &lt;init&gt;

`OfflineMessageSource()`

由一条消息中的 [QuoteReply](../-quote-reply/index.md) 得到的 [MessageSource](../-message-source/index.md).
此消息源可能来自一条与机器人无关的消息. 因此无法提供对象化的 `sender` 或 `target` 获取.
