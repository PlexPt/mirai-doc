ConstrainSingle

@SinceMirai("0.34.0") interface ConstrainSingle<out M : Message> : MessageMetadata

约束一个 MessageChain 中只存在这一种类型的元素. 新元素将会替换旧元素, 保持原顺序.

实现此接口的元素将会在连接时自动处理替换.
