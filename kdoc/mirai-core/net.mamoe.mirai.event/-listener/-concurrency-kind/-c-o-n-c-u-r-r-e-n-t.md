# CONCURRENT

`CONCURRENT`

并发地同时处理多个事件, 但无法保证 [onEvent](../on-event.md) 返回 [ListeningStatus.STOPPED](../../-listening-status/-s-t-o-p-p-e-d.md) 后立即停止事件监听.
