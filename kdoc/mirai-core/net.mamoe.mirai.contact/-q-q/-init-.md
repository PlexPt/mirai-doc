# &lt;init&gt;

`QQ()`

QQ 对象.

自 0.39.0 起 mirai 引入 [User](../-user/index.md) 作为 [Friend](../-friend/index.md) 和 [Member](../-member/index.md) 的父类,
以备将来支持仅 [Friend](../-friend/index.md) 可用的 API, 如设置备注.

所有 API 均有二进制兼容.

请根据实际情况, 使用 [Friend](../-friend/index.md) 或 [User](../-user/index.md) 替代.
